7 %

รับส่วนลดสูงสุด 7% ทันทีที่บิ๊กซี

• รับส่วนลด 7 % ทันที จากยอดใช้จ่ายผ่านบัตรฯ ที่บิ๊กซีช้อปปิ้งออนไลน์

• รับส่วนลด 5 % ทันที  จากยอดใช้จ่ายผ่านบัตรฯ ที่ห้างบิ๊กซี ซูเปอร์เซ็นเตอร์, บิ๊กซี เอ็กซ์ตร้า, บิ๊กซี มาร์เก็ต, มินิบิ๊กซี, บิ๊กซี ฟู้ดเพลส, บิ๊กซี ดีโป  และร้านขายยาเพรียว ทุกสาขา (เฉพาะสินค้าที่ร่วมรายการ)

*จำกัดส่วนลดสูงสุด 1,500 บาท / บัตร / เดือน เมื่อมียอดใช้จ่ายเต็มจำนวนสินค้าที่ร่วมรายการ และรวมยอดชำระเต็มจำนวนสินค้าในแผนกเครื่องใช้ไฟฟ้าด้วยบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด พร้อมสแกนบิ๊กพอยต์คิวอาร์โค้ด (QR Code) ด้านหลังบัตร (เฉพาะการรับสินค้าที่สาขา และชำระค่าสินค้าผ่านเว็บไซต์บิ๊กซี ช้อปปิ้งออนไลน์ (เพย์เม้นท์ เกตเวย์) เท่านั้น  กรณีชำระเงินปลายทางกับทางพนักงานส่งสินค้า จะไม่สามารถใช้ส่วนลดได้)

รับเพิ่มคะแนนสะสม AEON Happy Point สูงสุด 2,000 คะแนน

• ยอดใช้จ่ายทุกๆ 20 บาท ผ่านบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด รับ 1 คะแนนสะสม AEON Happy Point (ทุกช่องทางการใช้จ่าย)

• พิเศษ รับเพิ่มคะแนนสะสม AEON Happy Point สูงสุด 2,000 คะแนน / บัตร / รอบบัญชี  เมื่อมียอดการใช้จ่ายผ่านบัตรฯ ในหมวดหมู่ร้านอาหาร โรงแรม ช้อปปิ้งออนไลน์ และเมื่อใช้จ่ายเป็นสกุลเงินต่างประเทศ (MCC Code ตามที่บริษัทกำหนด)

• รับ AEON Happy Point X 3 เมื่อมียอดการใช้จ่ายสะสมต่อเดือน ตั้งแต่ 5,000 บาทขึ้นไป (คะแนนพิเศษ สูงสุด 500 คะแนน / บัตร / รอบบัญชี)

• รับ AEON Happy Point X 5 เมื่อมียอดการใช้จ่ายสะสมต่อเดือน ตั้งแต่ 10,000 บาทขึ้นไป (คะแนนพิเศษ สูงสุด 2,000 คะแนน / บัตร / รอบบัญชี)

• คะแนนสะสม AEON Happy Point สามารถใช้แลกรับโค้ดส่วนลดบิ๊กซี ได้ที่ AEON THAI MOBILE Application (1,000 คะแนน แลกรับโค้ดส่วนลดบิ๊กซี มูลค่า 100 บาท)

• บริษัทฯ สงวนสิทธิ์ในการยกเว้นการให้คะแนนสะสม และคำนวณยอดใช้จ่ายเพื่อร่วมรายการส่งเสริมการขายในทุกกรณี สำหรับรายการใช้จ่ายผ่านบัตรเครดิตอิออนทุกประเภทในหมวดกองทุนรวมทุกรายการ และประกันภัย/ชีวิต ตามระบบภายใต้ Code (MCC 5960, 6211, 6300, 9223)

0 %

แบ่งชำระผ่าน AEON HAPPY PLAN นานสูงสุด 10 เดือน

• แบ่งชำระรายเดือนกับ AEON Happy Plan ณ แผนกเครื่องใช้ไฟฟ้าที่ห้างบิ๊กซี ซูเปอร์เซ็นเตอร์ และบิ๊กซี เอ็กซ์ตร้าทุกสาขา ด้วยอัตราดอกเบี้ย 0%

• 0% นาน 10 เดือน ตั้งแต่ 10,000 บาทขึ้นไป / เซลล์สลิป

• 0% นาน 6 เดือน ตั้งแต่ 5,000 บาทขึ้นไป / เซลล์สลิป

แผนประกันอุบัติเหตุระหว่างเดินทาง

• รับสิทธิ์ประกันอุบัติเหตุระหว่างเดินทาง เมื่อชำระค่าเดินทางผ่านบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด รับความคุ้มครองครอบคลุมผู้ร่วมเดินทางทั้งคู่สมรส และบุตรอายุไม่เกิน 23 ปี วงเงินสูงสุดถึง 31,000,000 บาท / เหตุการณ์ สามารถศึกษารายละเอียดเพิ่มเติมได้ที่คู่มือสมาชิกบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด

• ยกเว้นค่าธรรมเนียมรายปี ปีแรก และปีถัดไป สำหรับบัตรหลัก และบัตรเสริม เมื่อใช้จ่ายผ่านบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด ที่ห้าง บิ๊กซี ซูเปอร์เซ็นเตอร์, บิ๊กซี เอ็กซ์ตร้า, บิ๊กซี มาร์เก็ต, มินิบิ๊กซี, บิ๊กซี ฟู้ดเพลส, บิ๊กซี ดีโป และร้านขายยาเพรียว ทุกสาขา หรือบิ๊กซีช้อปปิ้งออนไลน์อย่างน้อย 1 ครั้ง / ปี

สิทธิพิเศษจาก BIG POINT

• บิ๊กพอยต์สะสมได้ทุกบาทเมื่อช้อปที่บิ๊กซี ซูเปอร์เซ็นเตอร์, บิ๊กซี เอ็กซ์ตร้า, บิ๊กซี มาร์เก็ต, มินิบิ๊กซี, บิ๊กซี ฟู้ดเพลส, บิ๊กซี ดีโป, ร้านขายยาเพรียว ทุกสาขา และบิ๊กซี ช้อปปิ้งออนไลน์  พร้อมรับส่วนลดพิเศษจากร้านค้าชั้นนำเพียงแสดงใบเสร็จบิ๊กซีที่มีหมายเลขบิ๊กพอยต์ ศึกษารายละเอียดเพิ่มเติมได้ที่ bigpoint.bigc.co.th

ข้อกำหนด และเงื่อนไข

• สิทธิพิเศษสำหรับผู้ถือบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด (บัตรฯ) ที่ออกโดยบริษัท อิออน ธนสินทรัพย์ (ไทยแลนด์) จำกัด (มหาชน) (“บริษัทฯ”) เท่านั้น

เงื่อนไขการรับส่วนลดทันทีที่บิ๊กซี

• ส่วนลด 7% สำหรับการชำระค่าสินค้าผ่านทางออนไลน์ที่บิ๊กซีช้อปปิ้งออนไลน์ (เพย์เม้นท์ เกตเวย์) เท่านั้น

• ส่วนลด 5% สำหรับการชำระค่าสินค้าที่ห้างบิ๊กซี ซูเปอร์เซ็นเตอร์, บิ๊กซี เอ็กซ์ตร้า, บิ๊กซี มาร์เก็ต, มินิบิ๊กซี, บิ๊กซี ฟู้ดเพลส, บิ๊กซี ดีโป  และ ร้านขายยาเพรียว ทุกสาขา (เฉพาะสินค้าที่ร่วมรายการ)

• กรณีชำระเงินปลายทางกับทางพนักงานส่งสินค้า จะไม่สามารถใช้ส่วนลดได้

• กรณีชำระค่าสินค้าที่ห้างบิ๊กซี ผู้ถือบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด ต้องชำระค่าสินค้าที่ร่วมรายการ ด้วยบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ดพร้อมสแกนบิ๊กพอยต์คิวอาร์โค้ด (QR Code) ด้านหลังบัตรเดียวกันเท่านั้น

• จำกัดส่วนลดสูงสุด 1,500 บาท / บัตร / เดือน บัตรเสริมรับส่วนลดเพิ่มจากบัตรหลักสูงสุด 1,500 บาท / บัตร / เดือน (จำกัดจำนวน 1 บัตรเสริม / บัตรหลัก)

• รายการส่วนลดจะคำนวณจากยอดใช้จ่ายเต็มจำนวน และรวมยอดชำระเต็มจำนวนสินค้าภายในแผนกเครื่องใช้ไฟฟ้า โดยยอดใช้จ่ายคำนวณหลังจากหักคูปองส่วนลด หรือบัตรกำนัล สินค้าที่ไม่ร่วมรายการได้แก่ รายการสินค้าประเภทบุหรี่ สุรา เครื่องดื่มแอลกอฮอล์ เครื่องดื่มชูกำลังแบบแพ็ค กาแฟกระป๋องแบบแพ็ค นมผงดัดแปลงทั่วไปหรือสูตรต่อเนื่องสำหรับทารก และเด็กเล็ก บัตรกำนัล บัตรหรือรหัสเติมเงิน การชำระเงินตามใบแจ้งหนี้ การจองตั๋วออนไลน์ ค่ามัดจำร้านแว่นตาหรือร้านค้าภายในพลาซ่า ศูนย์อาหาร สินค้ากลุ่มยา เวชภัณฑ์ และสินค้าจดทะเบียนยา ธุรกรรมสินเชื่อเงินกู้ เช่าซื้อ หรือผ่อนชำระทุกประเภท และสินค้าอื่นตามที่บริษัทฯ กำหนด

• ยอดใช้จ่ายที่มีการคืนสินค้าและ / หรือยกเลิก ผู้ถือบัตรฯ จะได้รับยอดเงินคืนโดยเป็นยอดที่คำนวณหลังจากหักส่วนลดแล้วเท่านั้น

• บริษัทฯ สงวนสิทธิ์ในการคืนสิทธิ์ส่วนลด กรณีคืนสินค้าและ / หรือยกเลิกภายในเดือนที่ใช้จ่ายเท่านั้น

• เงื่อนไขส่วนลดค่าสินค้าเป็นไปตามที่บริษัท บิ๊กซี ซูเปอร์เซ็นเตอร์ จำกัด (มหาชน) กำหนด

• ส่วนลดเป็นสิทธิ์เฉพาะตัวของผู้ถือบัตรฯ ไม่สามารถโอน เปลี่ยน แลกเป็นเงินสด หรือสิทธิ์อื่น หรือบริการอื่นได้

เงื่อนไขการผ่อนชำระสินค้าผ่าน AEON Happy Plan

• ผู้ถือบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด สามารถผ่อนชำระเครื่องใช้ไฟฟ้า, โทรศัพท์มือถือ และสินค้าในแผนกเครื่องใช้ภายในบ้าน (ผ่านโปรแกรม AEON Happy Plan) ณ เคาน์เตอร์บริการสินเชื่อของอิออนในแผนกเครื่องใช้ไฟฟ้า ที่ห้างบิ๊กซี ซูเปอร์เซ็นเตอร์ และบิ๊กซี เอ็กซ์ตร้าทุกสาขา (เฉพาะสินค้าที่ร่วมรายการ)

• ดอกเบี้ย 0% นาน 10 เดือน เมื่อมียอดผ่อนชำระตั้งแต่ 10,000 บาทขึ้นไป / เซลล์สลิป

• ดอกเบี้ย 0% นาน 6 เดือน เมื่อมียอดผ่อนชำระตั้งแต่ 5,000 บาทขึ้นไป / เซลล์สลิป

• เงื่อนไข ข้อสัญญา ระเบียบ และหลักเกณฑ์ต่างๆ เป็นไปตามข้อกำหนดสำหรับบริการผ่อนชำระผ่านบัตรเครดิตอิออน (AEON Happy Plan) ที่บริษัทฯ กำหนด

• โปรดสอบถามรายละเอียดสินค้า และเงื่อนไขเพิ่มเติม ณ จุดขาย หรือที่เคาน์เตอร์บริการสินเชื่อของอิออนในแผนกเครื่องใช้ไฟฟ้าห้างบิ๊กซี ซูเปอร์เซ็นเตอร์ และบิ๊กซี เอ็กซ์ตร้าทุกสาขา

เงื่อนไขการรับคะแนนสะสม AEON Happy Point (20 บาท = 1 คะแนน) และคะแนนสะสมพิเศษ

• ยอดใช้จ่ายผ่านบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด ทุกๆ 20 บาท จะได้รับคะแนนสะสม (AEON Happy Point) 1 คะแนน (ยกเว้นการเบิกถอนเงินสดล่วงหน้า และค่าบริการอื่นๆ เช่น ชำระค่าธรรมเนียมล่าช้า ดอกเบี้ยจากหนี้ค้างชำระ ดอกเบี้ยการผ่อนชำระ ค่าธรรมเนียมการเบิกถอนเงินสดล่วงหน้า การชำระค่าไฟฟ้าผ่านบัตรเครดิต)

• คะแนนสะสมพิเศษสูงสุด 2,000 คะแนน/รอบบัญชี สำหรับยอดการใช้จ่ายผ่านบัตรฯ ในหมวดหมู่ร้านอาหาร โรงแรม (ภายใต้รหัสร้านค้า 5812-5814 และ 3501-3999 และ 7011 ในระบบวีซ่า มาสเตอร์การ์ด และ เจซีบี)  ช้อปปิ้งออนไลน์ (ยกเว้นบิ๊กซีช้อปปิ้งออนไลน์) และเมื่อใช้จ่ายเป็นสกุลเงินต่างประเทศ

• คะแนนสะสมพิเศษ จะแสดงในใบแจ้งยอดบัญชีบัตรเครดิต ภายใน 2 รอบบัญชีบัตรเครดิตนับจากรอบบัญชีที่มียอดใช้จ่าย

• ยอดค่าใช้จ่ายที่เกิดจากบัตรเสริมจะถูกคำนวนเข้าบัตรหลัก

• คะแนนสะสมทุก 1,000 คะแนน สามารถแลกเป็นโค้ดส่วนลดบิ๊กซีมูลค่า 100 บาท ได้ที่ AEON THAI MOBILE Application

• คะแนนสะสมจากบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด ไม่สามารถรวม หรือโอนคะแนนสะสมกับบัตรประเภทอื่นๆ ได้

• ท่านผู้ถือบัตรเครดิตบิ๊กซี แพลทินัม วีซ่า ที่มีสถานะปกติ เมื่อสมัคร และอนุมัติบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด คะแนนจะถูกโอนย้ายจากบัตรเครดิตบิ๊กซี แพลทินัม วีซ่า ไปยังบัตรเครดิตบิ๊กซี เวิลด์ มาสเตอร์การ์ด ภายใน 2 รอบบัญชีบัตรเครดิต

• สามารถตรวจสอบคะแนนที่โอนย้ายจากบัตรเครดิตบิ๊กซี แพลทินัม วีซ่า ได้ที่ www.aeon.co.th และ AEON THAI MOBILE Application หรือติดต่อศูนย์บริการลูกค้าบัตรบิ๊กซี

• โปรดตรวจสอบคะแนนสะสม วันหมดอายุ และรายละเอียดเพิ่มเติมในใบแจ้งการชำระเงิน หรือที่ www.aeon.co.th

• การคำนวณค่าใช้จ่ายของประเภทโรงแรม และร้านอาหาร บริษัทฯ ใช้ข้อมูลอ้างอิงตามที่ระบุใน Merchant Category Code (MCC Code) ตามระบบของ VISA, Mastercard และ JCB โดยจะคำนวณจากยอดใช้จ่ายตาม MCC Code ที่ระบุเท่านั้น ยอดใช้จ่ายนอกเหนือจาก MCC Code ที่ระบุจะถือว่าไม่เข้าเงื่อนไขการใช้จ่ายในหมวดต่างๆ ที่กำหนด

• บริษัทฯ สงวนสิทธิ์ในการเข้าร่วมรายการส่งเสริมการขายได้เฉพาะผู้ถือบัตรฯ ที่ยังคงสภาพสมาชิก และมีประวัติการชำระดีตามหลักเกณฑ์ของบริษัทฯ จนถึงวันที่ได้รับสิทธิพิเศษ และยกเว้นการมอบสิทธิพิเศษในกรณีดังต่อไปนี้ (1) มีการชำระเงินเข้าบัญชีบัตรเครดิตเกินวงเงินที่บริษัทฯ อนุมัติ (2) ใช้บัตรเครดิต หรือสินเชื่อเพื่อวัตถุประสงค์อันขัดต่อกฎหมาย หรือศีลธรรม เพื่อวัตถุประสงค์ทางธุรกิจ หรือใช้ในกิจการ หรือธุรกิจของตนเอง ครอบครัว เครือญาติ หรือบุคคลใกล้ชิด (3) ใช้บัตรเครดิต หรือสินเชื่อไม่เป็นไปตามข้อกำหนด และเงื่อนไขตามที่บริษัทฯ กำหนด หรือเงื่อนไขของรายการส่งเสริมการขายนี้ หากพบภายหลังว่าผู้ถือบัตรขาดคุณสมบัติ บริษัทฯ สงวนสิทธิ์ในการเรียกคืนสิทธิพิเศษ

• บริษัทฯ มิได้มีส่วนเกี่ยวข้องกับสินค้า และ / หรือบริการ หากพบความชำรุดบกพร่อง กรุณาติดต่อผู้ผลิตโดยตรง

• เงื่อนไขของบัตรฯ โค้ดส่วนลดบิ๊กซี เป็นไปตามที่บริษัท บิ๊กซี ซูเปอร์เซ็นเตอร์ จำกัด (มหาชน) กำหนด

• สิทธิพิเศษเป็นสิทธิ์เฉพาะตัวของผู้ถือบัตรฯ ไม่สามารถโอน หรือแลกเปลี่ยนเป็นเงินสด หรือสิทธิประโยชน์อื่นได้

• บริษัทฯ สงวนสิทธิ์ในการเปลี่ยนแปลงข้อกำหนด เงื่อนไข วิธีการ และสิทธิประโยชน์ใดๆ ภายใต้รายการส่งเสริมการขายนี้ โดยจะประกาศทางเว็บไซต์ www.aeon.co.th กรณีมีข้อพิพาทคำตัดสินของบริษัทฯ ถือเป็นที่สุด

สิทธิพิเศษจาก BIG POINT

• บิ๊กพอยต์สะสมได้ทุกบาทเมื่อช้อปที่บิ๊กซี ซูเปอร์เซ็นเตอร์, บิ๊กซี เอ็กซ์ตร้า, บิ๊กซี มาร์เก็ต, มินิบิ๊กซี, บิ๊กซี ฟู้ดเพลส, บิ๊กซี ดีโป, ร้านขายยาเพรียว ทุกสาขา และบิ๊กซีช้อปปิ้งออนไลน์  พร้อมรับส่วนลดพิเศษจากร้านค้าชั้นนำเพียงแสดงใบเสร็จบิ๊กซีที่มีหมายเลขบิ๊กพอยต์ ศึกษารายละเอียดเพิ่มเติมได้ที่ bigpoint.bigc.co.th

ข้อกำหนด และเงื่อนไข

• สิทธิพิเศษสำหรับผู้ถือบัตรเครดิตบิ๊กซี แพลทินัม วีซ่า (บัตรฯ) ที่ออกโดยบริษัท อิออน ธนสินทรัพย์ (ไทยแลนด์) จำกัด (มหาชน) (“บริษัทฯ”) เท่านั้น


* เงื่อนไขการรับส่วนลดทันทีที่บิ๊กซี

• ส่วนลด 3% สำหรับการชำระค่าสินค้าที่ห้างบิ๊กซี ซูเปอร์เซ็นเตอร์, บิ๊กซี เอ็กซ์ตร้า, บิ๊กซี มาร์เก็ต, มินิบิ๊กซี, บิ๊กซี ฟู้ดเพลส, บิ๊กซี ดีโป, ร้านขายยาเพรียว ทุกสาขา และบิ๊กซีช้อปปิ้งออนไลน์ (เพย์เม้นท์ เกตเวย์) (เฉพาะสินค้าที่ร่วมรายการ)

• กรณีชำระเงินปลายทางกับทางพนักงานส่งสินค้า จะไม่สามารถใช้ส่วนลดได้

• กรณีชำระค่าสินค้าที่ห้างบิ๊กซี ผู้ถือบัตรเครดิตบิ๊กซี แพลทินัม วีซ่า ต้องชำระค่าสินค้าที่ร่วมรายการ ด้วยบัตรเครดิตบิ๊กซี แพลทินัม วีซ่า พร้อมสแกนบิ๊กพอยต์คิวอาร์โค้ด (QR Code) ด้านหลังบัตรเดียวกันเท่านั้น

• จำกัดส่วนลดสูงสุด 1,000 บาท / บัตร / เดือน บัตรเสริมรับส่วนลดเพิ่มจากบัตรหลักสูงสุด 1,000 บาท / บัตร / เดือน (จำกัดจำนวน 1 บัตรเสริม / บัตรหลัก)

• รายการส่วนลดจะคำนวณจากยอดใช้จ่ายเต็มจำนวน และรวมยอดชำระเต็มจำนวนสินค้าภายในแผนกเครื่องใช้ไฟฟ้า โดยยอดใช้จ่ายคำนวณหลังจากหักคูปองส่วนลด หรือบัตรกำนัล สินค้าที่ไม่ร่วมรายการได้แก่ รายการสินค้าประเภทบุหรี่ สุรา เครื่องดื่มแอลกอฮอล์ เครื่องดื่มชูกำลังแบบแพ็ค กาแฟกระป๋องแบบแพ็ค นมผงดัดแปลงทั่วไปหรือสูตรต่อเนื่องสำหรับทารก และเด็กเล็ก บัตรกำนัล บัตรหรือรหัสเติมเงิน การชำระเงินตามใบแจ้งหนี้ การจองตั๋วออนไลน์ ค่ามัดจำร้านแว่นตาหรือร้านค้าภายในพลาซ่า ศูนย์อาหาร สินค้ากลุ่มยา เวชภัณฑ์ และสินค้าจดทะเบียนยา ธุรกรรมสินเชื่อเงินกู้ เช่าซื้อ หรือผ่อนชำระทุกประเภท และสินค้าอื่นตามที่บริษัทฯ กำหนด

• ยอดใช้จ่ายที่มีการคืนสินค้าและ / หรือยกเลิก ผู้ถือบัตรฯ จะได้รับยอดเงินคืนโดยเป็นยอดที่คำนวณหลังจากหักส่วนลดแล้วเท่านั้น

• บริษัทฯ สงวนสิทธิ์ในการคืนสิทธิ์ส่วนลด กรณีคืนสินค้าและ / หรือยกเลิกภายในเดือนที่ใช้จ่ายเท่านั้น

• เงื่อนไขส่วนลดค่าสินค้าเป็นไปตามที่บริษัท บิ๊กซี ซูเปอร์เซ็นเตอร์ จำกัด (มหาชน) กำหนด
